Description - Overview

In factory, workers always do same operation below:

  • Start Machine
  • Stop Machine
  • Repair Machine
  • Supply Material
Managers need to time for these work.

Purpose - Overview

Their needs as follows:

  • Auto time for start & stop machine.
  • Time for manual event trigger, such as repair & supply service.
  • Event trigger drive ring & light alarm.
So it is a Production Monitoring System using IoT Technology. Let's just call it "PMS".

Prototype Design - Overview

Here is my IoT Basic Model Definition. It has 5 layouts. From top to bottom, they are:

  • Application Service(应用服务层)
  • Data Service(数据服务层)
  • Communication(网络通讯层)
  • Sensor(感知层)
  • Physics(物理层)

Prototype Design - Overview

Based on my model, I designed this prototype architecture. PMS include 4 parts:

  • Sensor Data Collector(SDC)
  • Sensor Data Hub(SDH)
  • Data Filter & Persistence(DFP)
  • Information Visualization

Original System Implementation - Overview

  • Based on Arduino Ethernet implements SDC.
  • Implement SDH & DFP in Google Go.
  • Data stored in MariaDB with manual table partition.
  • Visualize Information partly & Alarm are implemented by LED Displayer drived RaspberryPi v2.
Their communication use socket, 0mq/nanomsg and websocket.

Readiness for Migrating to Azure IoT

Focus Core concepts for IoT.

  1. Events Hub
  2. Azure Storage
  3. Service Bus
  4. Notification Hub
  5. PowerBI

How to do it

  1. Events Hub replace SDH.
  2. Cloud Service Worker Role replace DFP.
  3. Azure SQL used for matedata management.
  4. Block Blob Storage replace MariaDB with manual partition what used for time series data store.
  5. Service Bus/Notification Hub replace 0mq PubSub.
  6. PowerBI will support better visualization.

Advantages vs. DisAdvantages

  • Advantages

    1. 充分利用Azure现有组件服务
    2. 缩短开发周期
    3. 提供高可靠性及可用性
    4. 便于维护管理

  • Disdvantages

    1. Arduino配置有限，不足以直接用于Events Hub
    2. 客户网络不稳定导致数据积压和丢失

How to prefact it - Advantages vs. DisAdvantages

  • Focus 2 points

    1. Arduino日志文件式暂存
    2. 部署本地缓冲服务器Normal PC
